# Demo 02: 表面纹理力渲染

## 力渲染模式
**表面纹理 (Surface Texture)** — 通过力的时空变化模拟不同材质

## 原理
在基础法向力之上叠加空间调制，当探针在表面上滑动时，
力的变化传递材质信息：

- **高低摩擦**: 改变切向力/法向力比值 (μ)
- **软硬差异**: 改变刚度系数 k (solref 参数)
- **凹凸纹理**: 在法向力上叠加空间周期性调制

## 测试材质

| 材质 | MuJoCo friction | MuJoCo solref | 力特征 |
|------|----------------|---------------|--------|
| 冰面 | 0.05 | 0.002, 1.0 | 极低切向力，高刚度 |
| 木板 | 0.4 | 0.003, 1.0 | 中等摩擦，稳定法向力 |
| 橡胶 | 1.5 | 0.008, 0.8 | 高摩擦，可见形变 |
| 海绵 | 0.6 | 0.04, 0.5 | 深侵入，大阻尼 |
| 钢铁 | 0.3 | 0.001, 1.2 | 极硬，近零侵入 |
| 果冻 | 0.2 | 0.05, 0.3 | 深侵入+高阻尼 |
| 砂纸 | 1.8 | 0.002, 1.0 | 极高摩擦 |
| 瓦楞 | 0.5 | 0.003, 1.0 | 周期性法向力变化 |

## 人体感知通道
- **SA-II**: 感知滑动方向和阻力大小 → Touch X ✅
- **FA-I**: 感知表面微结构振动 → Touch X ⚠️ 仅通过位移间接传递
- **SA-I**: 感知静态压力分布 → Touch X ❌ 无皮肤接触面

## Touch X 能力评估
⚠️ **有限支持** — Touch X 可渲染宏观力差异（软/硬/滑/涩），
但无法提供真实触觉纹理（皮肤振动），仅通过动觉通道间接传递。
闭眼测试时用户可以区分冰面 vs 橡胶（通过阻力），但无法区分砂纸 vs 木板（需要皮肤触觉）。
