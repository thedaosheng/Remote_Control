# Demo 03: 刚度墙（软硬碰撞）

## 力渲染模式
**可变刚度碰撞 (Variable Stiffness Contact)**

## 原理
通过改变弹簧常数 k 模拟不同硬度的物体。
- 低 k (50-200 N/m) → 软物体（海绵、橡皮泥）
- 中 k (500-1500 N/m) → 中等物体（木材、塑料）
- 高 k (2000-3500 N/m) → 硬物体（金属、陶瓷）

Touch X 最大可渲染刚度约 3.5 N/mm (3500 N/m)，受限于力输出极限。

## MuJoCo 实现
通过 `solref` 和 `solimp` 参数控制：
- `solref[0]` = 时间常数：越小越硬（0.001=钢，0.05=海绵）
- `solref[1]` = 阻尼比：1.0=临界阻尼
- `solimp` = [min_imp, max_imp, width]：控制侵入特性

## 人体感知
- **SA-I (Merkel)**: 主通道 — 感知物体硬度
- **SA-II (Ruffini)**: 辅通道 — 感知反力方向

✅ **完全支持** — 刚度渲染是 Touch X 最核心的能力
