# Demo 07: 磁吸效果

## 力渲染模式
**磁性吸附 (Magnetic Snap)** — 靠近目标时自动吸附

## 原理
在目标点附近产生吸引力（力随距离增大而减小）：
```
F = (strength / r²) · r̂  （指向目标）
```
超出 snap_radius 后力为零，实现"吸附区"效果。

## 应用场景
- 虚拟装配中的对齐引导
- CAD 中的捕捉点
- 手术中的目标定位辅助
- 遥操作中的精准对位

## Touch X 评估
✅ **完全支持** — 吸引力是纯位置相关力，Touch X 可精确渲染。
需注意力饱和（探针极近目标时限幅到 3.3N）。
