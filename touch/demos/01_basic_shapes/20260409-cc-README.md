# Demo 01: 基础几何体力渲染

## 力渲染模式
**刚度墙 (Stiffness Wall)** — Touch X 最基础的力渲染模式

## 原理
当 Touch 笔尖（SCP, Surface Contact Point）穿过虚拟几何体表面时，
产生与侵入深度成正比的法向恢复力：

```
F = k × penetration × n_hat
```

其中 `k` 为刚度系数，`penetration` 为侵入深度，`n_hat` 为表面法向量。

## 测试几何体
| 几何体 | MuJoCo 类型 | 尺寸 | 测试目的 |
|--------|-----------|------|---------|
| 平面 | plane | 无限大 | 法向力基准 |
| 球体 | sphere | Ø50mm | 曲面法向量方向变化 |
| 立方体 | box | 50×50×50mm | 棱边力突变 |
| 圆柱体 | cylinder | Ø40×60mm | 柱面切向滑动 |
| 锥体 | 多cylinder近似 | 底Ø50mm | 尖端接触+斜面 |

## 人体感知通道
- **SA-II (Ruffini)**: 主要通道 — 感知力的方向和大小
- **SA-I (Merkel)**: 辅助通道 — 感知侵入深度（静态压力）

## Touch X 能力评估
✅ **完全支持** — 这是 Touch X 的核心能力，3DOF 力输出足以渲染所有凸面法向力。
