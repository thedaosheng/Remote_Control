## 3. 能力边界总结

### ✅ Touch X 能做到的（动觉/力觉范畴）

| 能力 | 说明 | 感知通道 |
|------|------|---------|
| **刚度渲染** | 硬/软物体碰撞，力与侵入深度成正比 | SA-I, SA-II |
| **形状感知** | 通过表面法向力感知凸面几何体轮廓 | SA-II |
| **摩擦力** | 表面滑动时的切向阻力 | SA-II |
| **弹性交互** | 弹簧-阻尼系统，弹性物体操控 | SA-I, SA-II |
| **粘滞力** | 在粘稠介质中的运动阻力 | SA-II |
| **力场引导** | 虚拟弹簧/磁吸/引力引导用户运动 | SA-II |
| **碰撞反馈** | 硬碰撞的瞬态冲击力 | FA-II (间接) |
| **重量模拟** | 通过持续下拉力模拟物体重量 | SA-I |

### ❌ Touch X 做不到的（触觉/皮肤感知范畴）

| 缺失能力 | 说明 | 需要的技术 |
|---------|------|----------|
| **精细纹理** | 皮肤表面的微观粗糙度 | 皮肤振动器 (VCA/LRA) |
| **分布式压力** | 多点接触的压力分布 | 触觉阵列 (tactile array) |
| **温度反馈** | 冷/热感知 | Peltier 模块 |
| **皮肤拉伸** | 真实的切向皮肤变形 | 皮肤拉伸器 |
| **滑动检测** | 物体在手指上的滑动 | 指尖传感器 |
| **多指交互** | 双指捏取的力反馈 | 外骨骼手套 |

### ⚠️ 需要 VCA/LRA 补充的通道

- **FA-I (Meissner, 10-50Hz)**: Touch X 可通过力的周期性变化间接激活，但效果远弱于直接皮肤振动
- **FA-II (Pacinian, 40-400Hz)**: 电机带宽可能无法支持 >200Hz 的高频振动，需外部振动器
- **SA-I (Merkel)**: 单点力无法提供空间分布信息，需要触觉阵列

## 5. 运行指南

### 环境准备
```bash
pip install mujoco numpy matplotlib scipy
```

### Demo 运行命令

```bash
# 进入项目目录
cd ~/teleop/touch/mujoco_sim

# 1. 生成模拟 Touch 轨迹数据
python3 20260409-cc-mock_touch_data.py
# → 输出: screenshots/trajectory_*.png + trajectory_data/*.npy

# 2. 基础几何体碰撞力测试
python3 20260409-cc-touch_probe_basic.py
# → 输出: screenshots/demo_01_*_force.png, demo_01_*_stiffness.png

# 3. 表面纹理力渲染测试
python3 20260409-cc-touch_probe_textures.py
# → 输出: screenshots/demo_02_texture_*.png, demo_02_perception_heatmap.png

# 4. 8 种力渲染模式可视化
python3 20260409-cc-force_visualizer.py
# → 输出: screenshots/demo_03-10_*.png, demo_all_8_force_modes.png
```

### 一键运行全部 demo
```bash
cd ~/teleop/touch
bash 20260409-cc-run_all.sh
# 或者逐个运行上面的 4 个命令
```

### 无头运行（无显示器）
所有脚本使用 `matplotlib.use('Agg')` 后端，无需 X11/显示器即可运行。

### 注意
截图文件在首次运行 demo 后生成到 `screenshots/` 目录。
如果截图目录为空，请手动运行 `bash touch/20260409-cc-run_all.sh`。
